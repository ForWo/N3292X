This folder contains N3292x non-OS SDK and Linux BSP. 

\Doc\
	- contains some refernce documents 

\Utility\
	- contains some N3292x related utilities such as TurboWriter, AutoWriter, H264 Tool, etc. 

\Non-OS\
	- contains non-OS SDK (ADS 1.2) 

\Non-OS_KEIL\
	- contains non-OS SDK (KEIL) 

\Linux_BSP2.6,35\ 
	- contains the Linux 2.6.35 BSP and GNU tool chain 

\Linux\
	- contains the patch files and recent update as well as history 
	- contains some native application such as MiniGUI and MPlayer
